# IT Metrics Dashboard

## Overview

A Business Analysis portfolio project focused on improving IT Service Management (ITSM) visibility through KPI dashboards and ServiceNow integration.

## Business Problem

The IT Service Desk relied on manual Excel reporting, resulting in delayed visibility into SLA compliance, MTTR, and team performance.

## Proposed Solution

A real-time dashboard integrating with ServiceNow to provide:

- MTTR Tracking
- SLA Compliance Monitoring
- Incident Volume Analysis
- Team Performance Metrics
- Reporting Automation

## Business Objectives

- Reduce MTTR
- Improve SLA Compliance
- Eliminate Manual Reporting
- Enable Data-Driven Decision Making

- Non-Functional Requirements
